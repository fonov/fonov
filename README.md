# FonovTest

Тест для проверки iPhone перед покупкой. Бесплатный, без рекламы.

## Быстрый старт

```bash
cd fonov-app
npm install
npm run dev
```

Подробная документация для разработчиков: [fonov-app/README.md](fonov-app/README.md)

## Что нового (v2.0)

- Миграция на Vite + React 18 + TypeScript
- Поддержка iPhone XS — iPhone 17 (+ placeholder iPhone 18 / Fold)
- Новые тесты: Battery Health, USB-C, MagSafe, Action Button, Center Stage
- PDF-отчёт, быстрая проверка, тёмная тема
- 5 языков: EN, RU, UK, DE, TR
- PWA, CI/CD, E2E-тесты
